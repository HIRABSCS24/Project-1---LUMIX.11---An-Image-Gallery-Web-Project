# Lumix.11 — HD Wallpaper Gallery
A responsive wallpaper gallery website built 
from scratch using HTML, CSS and JavaScript.

# Features
-  Fully Responsive Design
-  4 Categories (Nature, Space, Cartoon, Aesthetic)
-  Download Functionality
- ❤️ Heart Toggle Feature
-  Dark Theme with Glass UI Effect
-  Hamburger Side Menu

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📂 Pages
- Home Page
- About Page
- Contact Page
- Nature / Space / Cartoon / Aesthetic Pages
